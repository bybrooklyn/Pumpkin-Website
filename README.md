# Pumpkin Site

a boring ai refactor because thats cool. HTML+CSS —> Astro+TS

## Commands

```sh
bun install
bun run dev
bun run build
bun run preview
```

Static assets and legacy public CSS paths live in `public/`. Shared page chrome lives in `src/layouts/` and `src/components/`; browser behavior lives in `src/scripts/`.
